# プロジェクト概要
2024.11.11-2024.11.27（17日間）

## 企画背景
近年、金融市場のデジタル化によりオンラインバンキングとフィンテックサービスが急速に普及しています。しかし、複雑なインターフェース、難解な金融用語、法的制限などにより、10代の若者の銀行サービス利用は依然として限定的です。そのため、銀行は将来の主要顧客層である10代を獲得するために様々な戦略を展開しています。KB国民銀行の'KBスターティーンズ'、カカオバンクの'記録通帳'、トスの'ティーンズ'サービスなどがその代表例です。このような状況の中、10代の目線に合わせた、よりアクセスしやすく理解しやすい金融サービスプラットフォームの開発の必要性が高まっています。
参考記事：https://www.kmib.co.kr/article/view.asp?arcid=0020032256


## 目標
未成年者と金融初心者を対象としたユーザーフレンドリーな金融サービスを提供するウェブアプリケーションの開発


## サービス名
**MDBANK**
：未成年顧客向けの金融推奨サービス（AI活用）

## 主要機能
**- AI資産タイプ分析および推奨機能**
  - ChatGPT APIを利用した資産タイプ分析サービス
  - 取引履歴を分析し、かわいいキャラクターを用いてタイプを提示
  - タイプに合った商品の提案
  - 
**- 振込および取引履歴機能**
  - 農協API：振込および取引履歴サービス
  - 農協振込APIで振込を行い、取引履歴APIで取引履歴機能を実装
  - 
**- 金融商品比較**
  - 金融監督院の定期預金商品APIと積立預金商品APIを利用した商品比較
  - Vue.jsでソート（sort）およびデータ表示
  - 預金・積立、預金、入出金別に利息順で商品ソート照会可能
  - localeCompare()を通じて商品名順ソートを実装
  - 
**- 周辺銀行支店検索サービス**
  - Kakao Maps APIを利用した支店検索サービス
  - デフォルトで「銀行」が検索ワードに含まれており、地域名のみ入力すれば該当地域の銀行が表示されるよう設定
      - 農協銀行本店を地図の中心に設定
  - 検索およびマーカー表示：
    検索結果に応じてマーカー生成および検索結果リスト表示
    マーカークリック時に詳細情報表示
    
**- 為替レート照会サービス**
  - 輸出入銀行為替レートAPIを利用した為替レート照会サービス
  - リアルタイム為替レート照会
  - リアルタイム為替レート照会時、期間内の為替レートを集計しヒストグラムを用いて為替変動推移を同時に表示

**- マイページ**
  - 個人情報照会
  - お気に入り照会および追加
  - プロフィール修正機能


## 使用技術スタック

**Python, Django, Vue.js, SQLite, git**


## 使用API
  金融監督院商品データAPI：金融商品情報
  カカオマップAPI：地図機能実装
  農協API：農協振込および取引履歴機能実装
  輸出入銀行為替レートAPI：為替情報
  ChatGPT API：ChatGPTを活用した資産分析機能実装


## トラブルシューティング
  - 会員登録および認証：
    dj-rest-authの使用によりパスワードハッシュ問題を解決。
    カスタマイズ：
      RegisterSerializerを継承し、追加フィールドおよびバリデーションを実装。
      get_cleaned_data()とsave()関数を活用。
  
  - API_KEY設定問題：
    .envファイルの=以降のスペースによりAPI認証失敗。
    スペース除去で解決。
  
  - 会員登録カスタマイズ：
    RegisterSerializerを活用し、拡張性を確保。
    既存のdj-rest-auth文書分析およびソースコード参照。
  
  - データ更新問題：
    CRUD後のリストと詳細画面データの非同期更新。
    詳細画面でバックエンドに再リクエストするようロジック修正。
  
  - 地図ロード問題：
    他のページに移動後、戻った際にロードされない問題。
    空値returnの部分を削除して解決。
  
  - 為替レート現在日付ロード問題：
    グラフ計算は現在日付で行われるが、為替レート値が以前にロードされたデータを読み込む問題。
    現在日付と読み込まれる値を同期させ、送金時の価格を対顧客電信売買相場に変更。
  
  - お気に入り追加問題：
    追加しても即時反映されず、リロード後に値が表示される問題。
    変数を同期させて解決。
  - AWS QuickSight利用問題：
    Amazon QuickSightのAIダッシュボード機能を活用した資産分析サービスを計画。
    機能実装のためAWS環境構築およびリソース設定を完了したが、Enterpriseプランの必要性による予算超過問題が発生。
    ChatGPTを利用した資産分析機能に方向転換。

# 프로젝트 개요
2024.11.11-2024.11.27(17일)

## 기획배경
최근 금융 시장의 디지털화로 온라인 뱅킹과 핀테크 서비스가 급속히 확산되고 있습니다. 그러나 복잡한 인터페이스, 어려운 금융 용어, 그리고 법적 제한 등으로 인해 10대들의 은행 서비스 이용이 여전히 제한적입니다. 이에 따라 은행들은 미래의 핵심 고객층인 10대를 유치하기 위해 다양한 전략을 펼치고 있습니다. KB국민은행의 'KB스타틴즈', 카카오뱅크의 '기록통장', 토스의 '틴즈' 서비스 등이 대표적입니다. 이러한 상황에서 10대들의 눈높이에 맞춘, 더욱 접근성 높고 이해하기 쉬운 금융 서비스 플랫폼 개발의 필요성이 대두되고 있습니다.

참고 기사 : https://www.kmib.co.kr/article/view.asp?arcid=0020032256

## 목표
미성년자와 금융 초보자들을 대상으로 사용자 친화적 금융 서비스를 제공하는 웹 애플리케이션 개발


## 서비스명
### MDBANK
** : 미성년자 고객 대상의 금융 추천 서비스 (AI활용)**


## 주요 기능
 **- AI 자산유형 분석 및 추천 기능**
  - chatGPT API를 이용한 자산유형 분석 서비스
  - 거래 내역을 분석해 귀여운 캐릭터를 이용해 유형제시
  - 유형에 맞는 상품 제시

 **- 이체 및 이체내역 기능**
   - 농협 API: 이체 및 이체내역 서비스
   - 농협 이체API로 이체하고, 거래내역API로 이체내역 기능 구현
     
 **- 금융 상품 비교**
  - 금융감독원 정기예금상품API와 적금상품 API를 이용한 상품비교
    Vue.js에서 정렬(sort) 및 데이터 표시
    - 예적금, 예금, 입출금별 이자순으로 상품정렬 조회 가능
    - localeCompare()를 통해 상품명 순 정렬 구현.

 **- 주변 은행 지점 찾기 서비스**
  - Kakao Maps API를 이용한 지점 찾기 서비스
  - 기본으로 은행이 검색할 단어에 포함이 되어있어 지역명만 입력하면 해당 지역의 은행 나오도록 설정
    - 농협은행 본점을 지도 중심으로 설정
      검색 및 마커 표시:
      검색 결과에 따라 마커 생성 및 검색 결과 목록 표시
      마커 클릭 시 상세 정보 표시

 **- 환율 조회 서비스**
   - 수출입은행 환율API를 이용한 환율 조회 서비스
   - 실시간 환율 조회
   - 실시간 환율 조회 시, 기간 내 환율을 모아 히스토그램을 이용한 환율의 변화추이 조회 함께 실행
     
 **- 마이페이지**
   - 개인 정보 조회
   - 즐겨찾기 조회 및 추가
   - 프로필 수정 기능

### 트러블슈팅

회원가입 및 인증
dj-rest-auth 사용으로 비밀번호 해싱 문제 해결.
커스터마이징:
RegisterSerializer 상속하여 추가 필드 및 유효성 검사 구현.
get_cleaned_data()와 save() 함수 활용.

API_KEY 설정 문제:
.env 파일에서 = 뒤 공백 문제로 API 인증 실패.
공백 제거로 해결.

회원가입 커스터마이징:
RegisterSerializer를 활용해 확장 가능성을 확보.
기존 dj-rest-auth 문서 분석 및 소스 코드 참조.

데이터 업데이트 문제:
CRUD 후 리스트와 디테일 화면 데이터 비동기 업데이트.
디테일 화면에서 백엔드 재요청하도록 로직 수정.

지도 로딩 문제:
다른 페이지로 갔다가 돌아올 때 로딩이 안되었던 문제.
빈값 return이 되는 부분을 지워주고 해결.

환율 현재날짜 로딩 문제:
그래프 계산은 현재날짜로 되나 환율 값이 불러와지는 값은 이전 로딩되었던 데이터가 불러와지는 문제
현재날짜와 불러지는 값 동일 시 시켜주고, 송금할 때 가격에서 매매기준율로 바꿔줌

즐겨찾기 추가 문제:
추가를 하는데 바로 값이 반응하지 않고 리로딩을 해줘야 값이 뜨는 문제
변수를 동일시 해주고 해결

AWS QuickSight 이용 문제:
Amazon QuickSight의 AI 대시보드 기능을 활용해 자산분석 서비스 진행계획.
기능 구현을 위해 AWS 환경 구축 및 리소스 설정을 완료했으나, Enterprise 요금제 필요성으로 인한 예산 초과 문제 발생.
chatGPT를 이용한 자산분석 기능으로 방향 수정.
